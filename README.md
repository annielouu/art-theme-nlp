# art-theme-nlp
This Streamlit application visualizes how themes in artwork have evolved through different periods. It is an interactive presentation of the findings from my previous analysis, found [here](https://github.com/annielouu/artwork_evolution_analysis).

## How to install and run
* Clone the repository: `git clone https://github.com/annielouu/art-theme-nlp.git`
* Install the required packages in requirements.txt: `pip install -r requirements.txt`
* Navigate to the project directory
* Use `conda activate myapp` to activate the environment you installed Streamlit in
* Run `streamlit run main.py`
* The app will pop out in your broswer.
